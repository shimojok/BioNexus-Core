# 🌍 **Planetary Health API（Agriculture Module）v1.0 — 完全統合設計**

以下は、Stripe の構造を農業インパクトに置き換えた **完全な OS 設計**です。

---

# 1. **AgriImpactIntent（中核オブジェクト）**

農業インパクトの「意図」を表す Stripe 的コア。

### 役割
- AGRIX データを受け取り  
- CO₂吸収、土壌再生、収量改善、水削減、MBT55改善を計算  
- PBPE価値を生成  
- 状態遷移（pending → computed → settled）を管理

### 最小 JSON（MVP）
```json
{
  "intent_id": "agi_intent_2026_0001",
  "field_id": "field_45",
  "inputs": {
    "soc_increase_t_per_ha": 0.42,
    "area_ha": 1.8,
    "yield_gain_percent": 12.3,
    "water_use_reduction_percent": 8.5,
    "mbt55_factor": 1.18
  },
  "computed": {
    "co2_sequestration_t": null,
    "soil_regeneration_index": null,
    "pbpe_value_jpy": null
  },
  "status": "pending"
}
```

---

# 2. **AgriImpactSource（AGRIX データ標準）**

圃場単位で必要な最小データ。

### Soil（必須）
- SOC before / after  
- 面積（ha）

### Crop（必須）
- 作物種  
- 収量 before / after  

### Water（必須）
- 水使用量 before / after  

### Microbiome（必須）
- MBT55 before / after  

これらを Intent の `inputs` にマッピングする。

---

# 3. **AgriImpactWebhook（MRV 自動化）**

Stripe Webhook の農業版。  
圃場で変化が起きた瞬間にイベントが発火。

### 7つの標準イベント
- soil.soc.updated  
- crop.yield.updated  
- water.use.updated  
- microbiome.mbt55.updated  
- agriimpact.intent.created  
- agriimpact.intent.computed  
- agriimpact.connect.settled  

これにより、**MRV（測定・報告・検証）が完全自動化**される。

---

# 4. **AgriImpactConnect（価値分配モデル）**

あなたが選んだ **40 / 40 / 20 モデル** を採用。

### 分配式
- 生産者：40%  
- PBPE基金：40%  
- プラットフォーム：20%  

### Connect JSON
```json
{
  "pbpe_value_jpy": 12800,
  "distribution": {
    "farmer_reward_jpy": 5120,
    "pbpe_fund_jpy": 5120,
    "platform_fee_jpy": 2560
  },
  "status": "settled"
}
```

Stripe Connect の「transfer.paid」に相当。

---

# 5. **AgriImpactCheckout（可視化 UI）**

Stripe Checkout の農業版。  
圃場のインパクトを 1 画面で理解できる。

### 表示内容
- CO₂吸収量  
- 土壌再生指数  
- 収量改善  
- 水使用削減  
- MBT55改善  
- PBPE価値  
- 価値分配（40/40/20）  

### Checkout JSON
```json
{
  "impact": {
    "co2_sequestration_t": 1.52,
    "soil_regeneration_index": 0.78,
    "yield_gain_percent": 12.3,
    "water_use_reduction_percent": 8.5,
    "mbt55_factor": 1.18
  },
  "value": {
    "pbpe_value_jpy": 12800,
    "farmer_reward_jpy": 5120,
    "pbpe_fund_jpy": 5120,
    "platform_fee_jpy": 2560
  }
}
```
