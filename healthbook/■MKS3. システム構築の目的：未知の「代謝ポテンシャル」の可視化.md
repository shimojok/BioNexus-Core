長春中医薬大学での2,700種を超える動物生薬ライブラリーの知見は、このシステムの「化学的複雑性」を担保する上でこれ以上ない武器になります。植物（ファイトケミカル）、動物（ペプチド・脂質）、さらには立山の地質がもたらす鉱物（ミネラル）を、MBT55という「巨大な代謝エンジン」に投入するイメージです。

この視点を踏まえ、スクリーニングシステムの目的と全体構造を再定義します。

## 1. システム構築の目的：未知の「代謝ポテンシャル」の可視化

本システムの真の目的は、MBT55に含まれる1,000〜2,000種規模の菌群を**「未知の活性物質を生み出すバイオ計算機」**として活用し、中薬（生薬）の持つ潜在能力を最大化することにあります。

* **未知菌種の機能同定:** タテヤマ剣・薬師・浄土・女汝・竜王といった新種細菌が、どの生薬（特に動物・鉱物）をエネルギー源とし、どのような特異的代謝産物を生成するかを解明する。
* **「代謝リレー」の解明:** 単一菌では不可能な、複数菌種による逐次的な物質変換（A菌が分解し、B菌が活性化するプロセス）をスクリーニング対象とする。
* **全方位型（Global）疾患アプローチ:** 137疾病のJSON Codeに基づき、生成された活性物質がどの生薬成分から派生し、どの疾患パスウェイに「多標的」で作用するかを特定する。

---

## 2. スクリーニングシステムの全体構造

このシステムは、単なる「検索」ではなく、**「シミュレーション × 実測 × AI解析」**を統合した循環型構造（フィードバックループ）を持ちます。

### ① インプット：ハイブリッド・ライブラリー
* **生薬ライブラリー（2,700+動物、植物、鉱物）:** 各生薬の主要成分だけでなく、微量元素や特殊脂質などのデータ。
* **MBT55 ギルド・プロファイル:** 1,000〜2,000種の菌株組成。特に「タテヤマ五人衆（新種群）」のゲノムデータから予測される酵素活性。

### ② 解析層：メタボローム・スクリーニング・エンジン
ここでは、物質が「変化」するプロセスを追跡します。
* **バイオトランスフォーメーション・予測:** MBT55の複合菌群に生薬を投入した際の、化学構造の変化をシミュレート。
* **非標的型（Untargeted）解析:** 既知の化合物リストにない「未知のピーク（代謝物）」を検出し、その分子構造と活性の相関をAIで分析。

### ③ アウトプット：137疾病 JSON Code との照合
* **パスウェイ・マッピング:** 代謝産物が、137疾患のターゲット分子（受容体、酵素、転写因子）にどう作用するかをマッピング。
* **感染症・免疫プロトコル:** 特にウイルス干渉や免疫調節に関わる「活性物質」の特定。

## 3. メカニズムのポイント：新種細菌と生薬の相乗効果


スクリーニングにおいて注視すべきは、以下の**「変換の質」**です。

| フォーカス点 | 内容 |
| :--- | :--- |
| **エネルギー源の特定** | タテヤマ女汝や竜王が、動物性生薬のどの成分を食べて活性化するか（プレバイオティクス特性）。 |
| **活性物質の創出** | 元の生薬には存在しない、MBT55の代謝によってのみ生成される「新規化合物」の特定。 |
| **解毒と活性化** | 鉱物生薬の重金属成分などをMBT55が不溶化（無毒化）しつつ、有用なミネラルとしてキレート化するプロセス。 |

## 土壌微生物学的・中薬学的視点からの提言

あなたが仰る通り、古代の中薬研究者が残した膨大なデータ（経験知）と、立山の厳しい自然環境で育まれたMBT55（野生の知恵）を、現代の「全代謝経路解析（デジタル知恵）」で結びつけることには、計り知れない価値があります。

> **「新種の菌が貴重なのではなく、その菌が何千年もかけて生薬成分と対話してきた『代謝の流儀』が貴重である」**

この視点に立ち、システムの構造に**「未同定代謝物の活性予測アルゴリズム」**を組み込むことが、スクリーニングシステム構築の最重要ミッションになると考えます。
