# 🌾 AgriImpactCheckout（可視化 UI）  
## **目的：農業インパクトを “Stripe Checkout のように一瞬で理解できる UI” にする**

Checkout は **UI を自動生成するためのデータ契約** です。  
Stripe が「金額・通貨・商品名」を渡すと決済 UI を自動生成するように、  
AgriImpactCheckout は **インパクト値を渡すと農業インパクト UI を自動生成**します。

---

# 🌱 **AgriImpactCheckout のデータ契約（MVP）**

以下が Checkout に渡すべき **最小 JSON** です。


# 🌍 **Checkout UI に表示される内容（Stripe Checkout の農業版）**

## 1. **CO₂吸収量（t）**
- 圃場がどれだけ炭素を固定したか  
- 国際炭素市場に接続可能な値  

## 2. **土壌再生指数（0〜1）**
- SOC・微生物・水分・MBT55 を統合した指標  
- 再生農業の “健康スコア”  

## 3. **収量改善（%）**
- 農家の経済価値に直結  
- PBPE価値の計算にも影響  

## 4. **水使用削減（%）**
- ESG の水資源評価に直結  

## 5. **MBT55 係数**
- 土壌微生物の代謝改善  
- AGRIX × MBT55 の統合ポイント  

## 6. **PBPE価値（円）**
- 圃場が生み出した “気候価値”  
- Connect による分配結果も表示  

## 7. **価値分配（生産者・PBPE基金・プラットフォーム）**
- 生産者：40%  
- PBPE基金：40%  
- プラットフォーム：20%  

Stripe Checkout の「料金明細」に相当。

- MBT55改善  

### ④ 価値分配（棒グラフ or 円グラフ）
- 生産者 40%  
- PBPE基金 40%  
- プラットフォーム 20%  

### ⑤ ESGレポートへのリンク
- 企業・自治体向け
