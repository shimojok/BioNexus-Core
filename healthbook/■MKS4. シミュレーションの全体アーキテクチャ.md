MBT55（NB菌）という巨大な「微生物ギルド」と、長春中医薬大学が誇る膨大な「生薬ライブラリー」を掛け合わせたスクリーニングシステムのシミュレーション設計を提案します。

この設計の肝は、個々の菌を単体で見るのではなく、**「特定の菌種が先遣隊として物質を分解し、後続の菌がそれを活性物質へ変える」という代謝リレー**をデジタル上で再現することにあります。

---

## 1. シミュレーションの全体アーキテクチャ

スクリーニングは以下の3つの階層（Layer）を順にシミュレートすることで、最適な「菌群×生薬」の組み合わせを絞り込みます。

### Layer A：基質分解シミュレーション（分解能の判定）
まずは、MBT55の主要菌種が持つ酵素群と、生薬成分の構造的相性を判定します。
* **タテヤマ浄土 (Bacillus sp.):** 強力なタンパク質・デンプン分解能力。動物性生薬（膠、胎盤、動物臓器など）の複雑な高分子をペプチドやオリゴ糖に断片化するプロセスを担当。
* **タテヤマ女汝 (Atopostipes sp.):** 配糖体の脱糖（デグリコシル化）に特化。植物性生薬のサポニンやフラボノイドを、吸収されやすいアグリコン（活性体）へ変換する。

### Layer B：代謝リレー・シミュレーション（新規化合物の生成）
断片化された成分を、嫌気性菌群がさらに深部代謝するプロセスです。
* **タテヤマ竜王 (Clostridium sp.):** 強力な還元・発酵能力。Layer Aで生成された中間体を、さらに脂質代謝物や短鎖脂肪酸、あるいは特殊な二次代謝産物へと変換。ここで「元の生薬には存在しない活性物質」が生まれます。

### Layer C：137疾病 JSON Code 照合（薬理効果の予測）
生成された最終代謝産物群を、疾病コードで定義されたターゲットにぶつけます。
* **例:** 生成された「未知のペプチドX」が、感染症コードの「ウイルス侵入受容体」に結合するかをスコアリング。

## 3. このシミュレーションがもたらす「未知」の発見

このシステムが完成すれば、単なる漢方の科学的根拠（エビデンス）の提示に留まらない、**「中薬の再構築（Re-Composition）」**が可能になります。

* **動物性生薬の再評価:** 長春中医薬大学の2,700種に及ぶ動物生薬は、MBT55のタテヤマ竜王（Clostridium）などの嫌気性代謝を受けることで、現代医学でも未知の「強力な抗炎症物質」や「神経再生因子」に化ける可能性があります。
* **土壌微生物の「知恵」の抽出:** 立山の過酷な環境を生き抜いた新種細菌群は、植物のファイトケミカルを「武器（抗菌物質）」や「シグナル（ホルモン様物質）」に作り変える術を知っています。その術を、人間の疾病治療に転用するのです。
