# 🌍 Planetary Health API — Climate Module v1.0  
**PBPE × Climate Finance の世界標準 API**

Stripe の構造をそのまま気候価値に移植します。

- PaymentIntent → **ClimateImpactIntent**  
- PaymentMethod → **ClimateSource**  
- Checkout → **ClimateImpactCheckout**  
- Webhook → **ClimateImpactWebhook**  
- Connect → **ClimateImpactConnect**  

これにより、  
**CO₂削減・炭素固定・PBPE価値生成・価値分配・金融接続**  
がすべて API で統一されます。

---

# 1. 🌱 **ClimateImpactIntent（気候インパクトの中核）**

Stripe の PaymentIntent に相当。

## 目的  
- CO₂削減・炭素固定・気候価値（PBPE）を自動計算  
- 状態遷移（pending → computed → settled）  
- 気候ファイナンスの基礎データを生成  

## MVP JSON
```json
{
  "intent_id": "cli_intent_2026_0001",
  "source_type": "agriculture" | "energy" | "transport" | "waste",
  "source_id": "field_45",

  "inputs": {
    "co2_reduction_t": 1.52,
    "co2_sequestration_t": 0.78,
    "energy_saving_kwh": 1200,
    "baseline_year": 2024
  },

  "computed": {
    "pbpe_value_jpy": null,
    "climate_score": null,
    "esg_value": null
  },

  "status": "pending"
}
```

# 2. 🌎 **ClimateSource（気候価値のデータ源）**

Stripe の PaymentMethod に相当。

PBPE は農業だけでなく、あらゆる気候行動を価値化できるため、  
**4つの標準ソース**を定義します。

## ① Agriculture（農業）  
- SOC  
- 土壌炭素  
- MBT55  
- AGRIX データ  

## ② Energy（エネルギー）  
- 再エネ導入  
- 省エネ  
- 電力削減  

## ③ Transport（移動）  
- EV 化  
- 燃費改善  
- 物流効率化  

## ④ Waste（廃棄物）  
- メタン削減  
- リサイクル  
- 堆肥化  

これらをすべて **ClimateImpactIntent の inputs に統合**できる。

Stripe が「カードでも銀行でも同じ API」で扱えるように、  
PH-API は **“どんな気候行動でも同じ API”** で扱える。

---

# 3. 🔔 **ClimateImpactWebhook（気候インパクトのイベント）**

Stripe Webhook の気候版。

## 7つの標準イベント
- climate.source.updated  
- climate.intent.created  
- climate.intent.computed  
- climate.connect.settled  
- co2.reduction.updated  
- co2.sequestration.updated  
- pbpe.value.generated  

これにより、  
**気候価値の MRV（測定・報告・検証）が完全自動化**される。

---

# 4. 💸 **ClimateImpactConnect（価値分配モデル）**

農業と同じ **40 / 40 / 20 モデル** を採用。

- 生産者・企業：40%  
- PBPE基金：40%  
- プラットフォーム：20%  

## JSON（MVP）
```json
{
  "pbpe_value_jpy": 9800,
  "distribution": {
    "actor_reward_jpy": 3920,
    "pbpe_fund_jpy": 3920,
    "platform_fee_jpy": 1960
  },
  "status": "settled"
}
```

Stripe Connect の「transfer.paid」に相当。

---

# 5. 📊 **ClimateImpactCheckout（可視化 UI）**

Stripe Checkout の気候版。

## 表示内容
- CO₂削減量  
- CO₂固定量  
- PBPE価値  
- ESGスコア  
- 価値分配（40/40/20）  
- 気候行動の詳細（農業・エネルギー・交通など）  

## JSON（MVP）
```json
{
  "impact": {
    "co2_reduction_t": 1.52,
    "co2_sequestration_t": 0.78,
    "climate_score": 0.82
  },
  "value": {
    "pbpe_value_jpy": 9800,
    "actor_reward_jpy": 3920,
    "pbpe_fund_jpy": 3920,
    "platform_fee_jpy": 1960
  }
}
```
