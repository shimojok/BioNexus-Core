## 2. PBPE Credits（Carbon / Biosecurity / Food Loss / Quality）正式仕様書（ドラフト）

### 2.1 共通仕様

- **単位:** 1 PBPE = 1 “standardized impact unit”
- **属性:**
    - `type`: `"carbon" | "biosecurity" | "food_loss" | "quality"`
    - `project_id`
    - `vintage_year`
    - `amount_pbpe`
    - `methodology_id`
    - `mrv_reference`（MBT‑Biosecurity‑Engine / Field Data / Third-party）

---

### 2.2 Carbon PBPE

- **対象インパクト：**
    - GHG削減（tCO₂e）
- **発行式：** [ \text{Carbon PBPE} = \text{Verified GHG Reduction (tCO₂e)} ]
- **主なソース：**
    - 廃棄物処理回避
    - 食品ロス削減
    - 土壌炭素隔離
    - 化学肥料削減
    - 家畜メタン削減
    - 稲作メタン削減
- **用途：**
    - 企業の Scope 1 / 2 / 3 オフセット
    - PBPE‑backed Bond の基礎指標

---

### 2.3 Biosecurity PBPE

- **対象インパクト：**
    - 微生物・病原体リスク低減
    - 生態系安定性向上
- **発行式（例）：** [ \text{Biosecurity PBPE} = f(\text{Risk Reduction Index}, \text{Coverage Area}, \text{Duration}) ]
- **データソース：**
    - MBT‑Biosecurity‑Engine（ハイパーサイクル安定度・リスク指数）
- **用途：**
    - 食品安全・パンデミックリスク低減の価値化
    - 保険・再保険モデルとの連携

---

### 2.4 Food Loss PBPE

- **対象インパクト：**
    - 食品ロス削減（t）
- **発行式：** [ \text{Food Loss PBPE} = \text{Food Loss Avoided (t)} \times \beta ] （(\beta) は tCO₂e換算＋食料安全保障係数）
- **データソース：**
    - 小売・流通のロス率
    - MBT55 による鮮度保持データ
- **用途：**
    - 小売・流通の Scope 3 削減
    - 食料安全保障プログラム

---

### 2.5 Quality PBPE

- **対象インパクト：**
    - 栄養価・機能性・鮮度・一貫性
- **発行式（例）：** [ \text{Quality PBPE} = \text{Nutrition Index Improvement} \times \text{Volume (t)} ]
- **データソース：**
    - MBT Nutrition Index（リコピン・ポリフェノール・ミネラル・抗酸化能・鮮度）
- **用途：**
    - 機能性食品・健康価値の定量化
    - 保険・医療費削減モデルとの接続

---

## 3. PBPE‑backed Bond の金融モデル（ドラフト）

### 3.1 基本構造

- **名称:** PBPE‑backed Climate & Health Bond
- **原資産:** PBPE 発行量（Carbon + Biosecurity + Food Loss + Quality）
- **投資家:** 機関投資家・開発銀行・ESGファンド
- **発行体:** PBPE Marketplace SPC / 政府系機関

---

### 3.2 クーポン設計（例）

[ \text{Coupon Rate} = r_0 + \alpha \cdot \frac{\text{PBPE}_{\text{actual}} - \text{PBPE}_{\text{floor}}}{\text{PBPE}_{\text{floor}}} ]

- ( r_0 ): ベースクーポン（例：2.0%）
- ( \alpha ): パフォーマンス連動係数（例：2.0）
- ( \text{PBPE}_{\text{floor}} ): 最低保証発行量
- ( \text{PBPE}_{\text{actual}} ): 実際の PBPE 発行量

→ **PBPE 発行量が増えるほど投資家の利回りが上がる “インパクト連動債”**

---

### 3.3 リスク・保全

- **リスク要因：**
    - プロジェクト実行リスク
    - MRV リスク
    - 市場価格リスク（PBPE価格）
- **保全手段：**
    - PBPE フロア保証
    - 政府・MDB の部分保証
    - インパクト保険（PBPE‑Insurance）

---

### 3.4 ダッシュボード表示との連動

- `/dashboard/finance/bonds` で：
    - Outstanding
    - Coupon
    - Linked PBPE Floor
    - Maturity
- 将来的に：
    - 実績 PBPE に応じたクーポン変動をリアルタイム表示

## 4. PBPE‑Dashboard（統合ビュー）の API 設計

PBPE‑Dashboard は **3つのバックエンド** からデータを集約：

- PBPE‑Marketplace API
- MBT‑Biosecurity‑Engine API
- PBPE‑Finance API

---

### 4.1 統合ダッシュボード用 API（例）

**ベース：`pbpe-dashboard` サービス側の API**

---

**GET `/planetary/summary`**

- **統合内容：**
    
    - GHG削減（Marketplace）
    - 生態系指数（Biosecurity）
    - PBPE発行量（Marketplace）
    - 経済価値（Finance）
- **レスポンス例：**
    

```json
{
  "ghg_reduction_tco2e_per_year": 510000000,
  "soil_health_index": 0.78,
```
