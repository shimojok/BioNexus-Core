# 🌍 **🅔 PBPE Rating Engine — Full Specification（完全版）**

PBPE Rating Engine は、PBPE 金融商品のリスクを  
**A / B / C / D の 4 段階で格付け**します。

格付けは以下の 5 つの要素から構成されます：

```
PBPE Rating = f(
  Intent Reliability,
  Data Quality,
  Regional Risk,
  PBPE Volatility Index (PVI),
  PBPE Default Probability (PDP)
)
```

---

# 1. **Intent Reliability Score（IRS）**

Planetary Health OS の Intent Layer の信頼性を評価する。

### **評価項目**

- MRV 自動化率（Webhook の発火率）
- データソースの透明性
- Intent の履歴（過去の安定性）
- 異常値の頻度
- Intent の再現性（同条件で同結果が出るか）

### **スコア式**

```
IRS = 0.4 * MRV_score
    + 0.3 * DataTransparency_score
    + 0.2 * Stability_score
    + 0.1 * Reproducibility_score
```

### **スコア範囲**

- 0–100
- 高いほど信頼性が高い

---

# 2. **Data Quality Score（DQS）**

Intent に入力されるデータの品質を評価する。

### **評価項目**

- AGRIX / ClimateSource / HealthBook のデータ完全性
- センサー精度
- サンプリング頻度
- 欠損値率
- 外れ値率

### **スコア式**

```
DQS = 100 - (MissingRate * 20 + OutlierRate * 30 + LowFrequencyPenalty)
```

---

# 3. **Regional Risk Score（RRS）**

地域固有のリスクを評価する。

### **評価項目**

- 気候リスク（干ばつ・洪水・高温）
- 政治リスク
- 経済リスク
- 規制リスク
- インフラリスク

### **スコア式**

```
RRS = 100 - (ClimateRisk * 0.3 + PoliticalRisk * 0.2 + EconomicRisk * 0.2 + RegulatoryRisk * 0.2 + InfraRisk * 0.1)
```

---

# 4. **PBPE Volatility Index（PVI）**

PBPE の変動性を評価する。

### **定義**

```
PVI = 標準偏差(PBPE_yield) / 平均(PBPE_yield)
```

### **分類**

- PVI < 0.10 → 低ボラティリティ
- 0.10–0.25 → 中ボラティリティ
- > 0.25 → 高ボラティリティ
    

---

# 5. **PBPE Default Probability（PDP）**

PBPE 金融商品の “価値未達リスク” を評価する。

### **定義**

```
PDP = 1 - (実績PBPE / 期待PBPE)
```

### **分類**

- PDP < 0.05 → 非常に低リスク
- 0.05–0.15 → 中リスク
- > 0.15 → 高リスク
    

---

# 6. **PBPE Composite Risk Score（PCRS）**

最終的な PBPE リスクスコア。

### **式**

```
PCRS = 0.30 * IRS
     + 0.25 * DQS
     + 0.20 * RRS
     + 0.15 * (1 - PVI_normalized)
     + 0.10 * (1 - PDP)
```

### **PVI 正規化**

```
PVI_normalized = min(PVI / 0.30, 1)
```

---

# 7. **PBPE Rating（A/B/C/D）への変換**

```
PCRS ≥ 85 → PBPE-A（最も安全）
PCRS ≥ 70 → PBPE-B（安定）
PCRS ≥ 55 → PBPE-C（リスクあり）
PCRS < 55 → PBPE-D（高リスク）
```
