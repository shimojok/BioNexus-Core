# 🩺 **HealthImpactIntent v1 — REST API 仕様（MVP）**

目的：  
HealthBook・MBT Probiotics・生活習慣データから、  
**健康改善 → 医療費削減 → PBPE価値 → 分配**  
を一括で計算する中核 API。

Stripe の PaymentIntent と同じ構造で、  
健康インパクトを “7 行で扱える” 抽象レイヤー。

---

# 1. POST /health/impact_intents  
### **目的：健康インパクト Intent を新規作成**

### Request Body（MVP）

```json
{
  "user_id": "user_123",

  "inputs": {
    "mbt55_before": 42,
    "mbt55_after": 58,

    "bmi_before": 27.5,
    "bmi_after": 24.8,

    "bp_before": "145/95",
    "bp_after": "128/82",

    "steps_per_day": 6500,
    "sleep_hours": 6.8,

    "program_duration_days": 90
  },

  "context": {
    "country": "JP",
    "age": 44,
    "sex": "male",
    "baseline_year": 2024
  }
}
```

### Response（201 Created）

```json
{
  "intent_id": "hlth_intent_2026_0001",
  "status": "pending",

  "inputs": {
    "mbt55_change": 16,
    "bmi_change": -2.7,

  },

}
```
