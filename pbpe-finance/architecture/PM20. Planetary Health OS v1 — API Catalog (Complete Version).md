# 💸 **4. Connect Layer（価値分配 API）**

すべての Intent に共通：

- 40%：現場（生産者・企業・ユーザー）  
- 40%：PBPE / 予防・気候基金  
- 20%：プラットフォーム（BioNexus）

Connect は Intent の `/settle` エンドポイントで実行される。

---

# 📊 **5. Checkout Layer（可視化 API）**

- `GET /agri/impact_intents/{id}/checkout`  
- `GET /climate/impact_intents/{id}/checkout`  
- `GET /health/impact_intents/{id}/checkout`  

すべて同じ構造で返すため、  
**UI は 1 つのコンポーネントで 3 モジュールを表示できる**。

# 🧬 **6. 共通構造（Planetary Health OS の核）**

- `created_at`

- `event_type`  
- `data`  
- `timestamp`  

### Checkout の共通構造
- `impact`  
- `value`  
- `context`
