# 1. **収量 × 品質 × 生産費削減 → 収益が構造的に向上する**

あなたのグラフは、MBT55 が：

- 収量：2 → 4.5
- 品質：2 → 6
- 栄養価：2 → 8
- 鮮度保持：-5 → -2
- 食品ロス：-5 → -3.5
- 廃棄：-5 → -4

という **多次元の改善**を示している。

これを経済式にすると：

---

## **利益 = 収量 × 単価 + 品質プレミアム − 生産費 − ロスコスト**

MBT55 によって：

### ✔ 収量が増える

→ 売上が増える

### ✔ 品質が上がる

→ プレミアム価格がつく

### ✔ 栄養価が上がる

→ 機能性食品としての価値が上がる

### ✔ 鮮度保持力が上がる

→ 食品ロスが減る → 利益が増える

### ✔ 廃棄物が減る

→ 廃棄コストが減る

### ✔ 生産費が最大84%削減

→ 利益率が劇的に改善する

# 2. **さび病防止 → 収量安定化 → 価格適正化**

コーヒー業界の最大の問題は：

- さび病
- 気候変動
- 土壌劣化

これらが **収量の不安定化 → 価格の乱高下** を引き起こしている。

MBT55 / AGRIX によって：

### ✔ さび病発生率が 50〜80% 低下

### ✔ 樹勢が回復

### ✔ 水分保持力が改善

### ✔ 幼木の生存率が上昇

### ✔ 収量が安定する

すると：

> **供給が安定する → 市場価格が安定する → 適正価格が形成される**

これは PBPE Marketplace の **PBPE Rating（A〜D）** にも反映される。

---

# 3. **PBPE Marketplace による “価格の適正化” の意味**

価格が適正化するとは：

- **安定供給による市場価格の乱高下の抑制**
- **高品質・高レジリエンスの農園が正当に評価される**
- **PBPE 値（炭素・土壌・健康・循環価値）が価格に上乗せされる**
- **金融商品（PBPE Coffee Resilience Bond）として裏付けが生まれる**

つまり：

> **コーヒーの価格が「科学的に裏付けられた価値」に基づいて決まるようになる。**

---

# 4. **PBPE Marketplace が生み出す “新しい収益源”**

従来のコーヒー農家の収入は：

- 収量 × 市場価格

だけだった。

PBPE Marketplace では：

### ✔ 収量収入

### ✔ PBPE（炭素・土壌・健康・循環）収入

### ✔ PBPE Bond / Credit の分配（40%）

### ✔ Scope3 削減価値の販売

### ✔ レジリエンス評価（PBPE Rating）によるプレミアム価格

が追加される。

つまり：

> **コーヒー農家は「コーヒーを売る」だけでなく  
> 「レジリエンス価値」「炭素価値」「健康価値」も売るようになる。**
