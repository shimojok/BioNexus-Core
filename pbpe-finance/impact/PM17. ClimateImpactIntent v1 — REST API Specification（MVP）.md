### 🌍 ClimateImpactIntent v1 — REST API 仕様（MVP）

**目的：**  
あらゆる気候行動（農業・エネルギー・交通・廃棄物など）から、  
CO₂削減・炭素固定・PBPE価値を一元的に計算し、資金循環につなげる中核 API。

---

## 1. POST /climate/impact_intents  
**目的：気候インパクト Intent を新規作成（PaymentIntent.create 相当）**

### Request Body（MVP）

```json
{
  "source_type": "agriculture", 
  "source_id": "field_45",

  "inputs": {
    "co2_reduction_t": 1.20,
    "co2_sequestration_t": 0.80,
    "energy_saving_kwh": 0,
    "baseline_year": 2024
  },

  "context": {
    "country": "JP",
    "region": "Chiba",
    "project_type": "regenerative_agriculture",
    "registry": "PBPE_INTERNAL"
  }
}
```

- **source_type:** `"agriculture" | "energy" | "transport" | "waste"`  
- **inputs:** CO₂削減・固定・エネルギー削減など  
- **context:** 国・地域・プロジェクト種別・レジストリ

### Response（201 Created）

```json
{
  "intent_id": "cli_intent_2026_0001",
  "status": "pending",

  "source_type": "agriculture",
  "source_id": "field_45",

  "inputs": {
    "co2_reduction_t": 1.20,
    "co2_sequestration_t": 0.80,
    "energy_saving_kwh": 0,
    "baseline_year": 2024
  },

  "computed": {
    "climate_score": null,
    "pbpe_value_jpy": null,
    "esg_value": null
  },

  "created_at": "2026-04-30T09:10:00"
}
```

---

## 2. GET /climate/impact_intents/{intent_id}  
**目的：Intent の最新状態を取得**

### Response（200 OK）

```json
{
  "intent_id": "cli_intent_2026_0001",
  "status": "computed",

  "computed": {
    "climate_score": 0.82,
    "pbpe_value_jpy": 9800,
    "esg_value": 0.75
  },

  "value_distribution": {
    "actor_reward_jpy": 3920,
    "pbpe_fund_jpy": 3920,
    "platform_fee_jpy": 1960
  }
}
```

---

## 3. POST /climate/impact_intents/{intent_id}/compute  
**目的：気候インパクト計算を実行**

### Response（200 OK）

```json
{
  "intent_id": "cli_intent_2026_0001",
  "status": "computed",

  "computed": {
    "climate_score": 0.82,
    "pbpe_value_jpy": 9800,
    "esg_value": 0.75
  }
}
```

---

## 4. POST /climate/impact_intents/{intent_id}/settle  
**目的：PBPE価値の分配（40/40/20）を実行**

### Response（200 OK）

```json
{
  "intent_id": "cli_intent_2026_0001",
  "status": "settled",

  "distribution": {
    "actor_reward_jpy": 3920,
    "pbpe_fund_jpy": 3920,
    "platform_fee_jpy": 1960
  },

  "settled_at": "2026-04-30T09:15:00"
}
```

- **actor_reward:** 行為主体（農家・企業・自治体など）  
- **pbpe_fund:** 気候価値基金  
- **platform_fee:** BioNexus / PH-OS 運営

---

## 状態遷移

- `pending` → `computed` → `settled`  
Agri と完全に同じパターンで、OS として一貫性を保つ。
