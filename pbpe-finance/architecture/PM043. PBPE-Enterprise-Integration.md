# 🌐 **Step 7：PBPE Marketplace — Enterprise Integration**

企業統合では、以下の 3 つを統合します：

1. **Scope 3（企業排出） → PBPE 自動変換**
2. **PBPE Finance（Credits / Bonds / Insurance）との連動**
3. **PBPE Dashboard v3 に企業データを統合表示**

つまり：

> **企業の環境負荷（Scope 3）を PBPE に変換し、  
> その PBPE を金融価値（Credits / Bonds / Insurance）に接続し、  
> ダッシュボードで “企業の地球価値” を可視化する。**

これが PBPE Marketplace の “企業導入版” です。
