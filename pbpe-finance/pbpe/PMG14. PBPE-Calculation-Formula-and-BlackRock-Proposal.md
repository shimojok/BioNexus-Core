# 📘 **② PBPE Rating（A〜D）算定式（完全版）**

PBPE Rating は、PBPE Marketplace における  
**金融商品（PBPE Bonds / Credits）・企業のScope3評価・投資判断**  
の基盤となる評価指標です。

PBPE Rating は以下の 5 つの指標から構成されます：

1. **IRS（Intent Reliability Score）**
2. **DQS（Data Quality Score）**
3. **RRS（Regional Risk Score）**
4. **PVI（PBPE Volatility Index）**
5. **PDP（PBPE Default Probability）**

---

# 🔶 **PBPE Rating の総合式**

[ PBPE\ Rating = w_1 IRS + w_2 DQS + w_3 (1 - RRS) + w_4 (1 - PVI) + w_5 (1 - PDP) ]

- 各スコアは 0〜1
- 重み ( w_1〜w_5 ) は合計 1
- RRS / PVI / PDP は「リスク」なので **1 − 値** を採用

---

# 🔧 **推奨重み（コーヒー産業向け）**

|指標|重み|理由|
|---|---|---|
|IRS|0.30|生産者の実行能力が最重要|
|DQS|0.20|データの信頼性は金融商品に必須|
|RRS|0.20|気候・病害リスクの地域差が大きい|
|PVI|0.15|PBPE の変動性を抑える|
|PDP|0.15|PBPE 不履行リスクを反映|

---

# 1. **IRS（Intent Reliability Score）**

[ IRS = \frac{Actual_Impact}{Expected_Impact} ]

- 収量改善
- 歩留まり改善
- 生産費削減
- SOC 増加
- GHG 削減

**MBT55 の実績値が高いほど IRS が上昇。**

---

# 2. **DQS（Data Quality Score）**

[ DQS = \frac{Verified\ Data}{Total\ Data} ]

- AGRIX センサー
- HealthBook
- 土壌データ
- PBPE Ledger の監査データ

**第三者検証が多いほど高スコア。**

---

# 3. **RRS（Regional Risk Score）**

[ RRS = f(ClimateRisk, DiseaseRisk, SoilRisk) ]

- 気温変動
- 降水量変動
- さび病リスク
- 土壌劣化リスク

**MBT55 導入により RRS は大幅に低下する。**

---

# 4. **PVI（PBPE Volatility Index）**

[ PVI = \sigma(PBPE\ Value) ]

- PBPE 値の変動性
- 収量の安定性
- 品質の安定性

**MBT55 により変動性が低下 → PVI が低くなる。**

---

# 5. **PDP（PBPE Default Probability）**

[ PDP = f(IRS, RRS, PVI) ]

- PBPE が予定通り発生しない確率
- さび病・気候変動・生産者能力が影響

---

# ⭐ **PBPE Rating のランク定義**

|ランク|スコア範囲|意味|
|---|---|---|
|**PBPE‑A**|0.85–1.00|高レジリエンス・高信頼性・低リスク|
|**PBPE‑B**|0.70–0.85|安定的で投資適格|
|**PBPE‑C**|0.55–0.70|リスクあり、改善余地大|
|**PBPE‑D**|0.00–0.55|高リスク、投資不適格|
